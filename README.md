# Mercurius Landing Page

Official landing page for **Mercurius Solutions** — an AI-powered virtual marketplace connecting homeowners with trusted local home service vendors in Southwest Florida (Cape Coral & Fort Myers).

## Live Site

- **Landing Page**: [https://mercuriusmarketplace.com](https://mercuriusmarketplace.com)
- **Vendor Application**: [https://applymercurius.com](https://applymercurius.com)

## Features

- Clean, modern design optimized for Southwest Florida homeowners and vendors
- Side-by-side call-to-action sections for homeowners and vendors
- Integrated Calendly booking for demos
- Waitlist form connected to Zapier
- Fully responsive and fast-loading

## Tech Stack

- HTML5 + Tailwind CSS
- Vanilla JavaScript
- Hosted on Vercel
- Connected to GitHub for continuous deployment

## Local Development

1. Clone the repository
2. Open the project folder in VS Code
3. Open `index.html` with **Live Server** extension (recommended)
4. Or simply open `index.html` directly in your browser

## Deployment

This project is automatically deployed to Vercel whenever changes are pushed to the `main` branch.

## Project Purpose

This landing page serves two main audiences:

- **Homeowners**: Can join the waitlist and book a demo
- **Vendors & Local Businesses**: Can apply to join the Mercurius platform
